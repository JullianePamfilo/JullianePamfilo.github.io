The Databases artifact showcases my understanding of data persistence, CRUD operations, and data integrity. By enhancing the Python-based database application, I focused on validation, error handling, and clear separation between data access and application logic. This artifact highlights my ability to work with structured data responsibly and securely, ensuring that applications behave predictably and safely when interacting with stored information.

Together, these artifacts demonstrate that I have met the core program outcomes of designing and evaluating computing solutions, applying appropriate data structures and algorithms, and effectively managing data.
